Mobclix
=======

Gideros plugin to allow the use of Mobclix adverts

Example:<br />
<pre lang="lua">
mobclix require "mobclix"<br>
<br>
mobclix:loadAd()<br>
mobclix:removeAd()<br>
adWidth = mobclix:getWidth()<br>
adHeight = mobclix:getHeight()<br>
xp = mobclix:getX()<br>
yp = mobclix:getY()<br>
mobclix:setX(xp)<br>
mobclix:setY(yp)<br>
mobclix:setPosition(xp, yp)<br>
mobclix:setVisible(vis)<br>
</pre>
